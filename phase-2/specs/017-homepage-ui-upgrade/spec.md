# TaskGenie Specification

## Project Overview
TaskGenie is a high-performance, AI-powered task management application designed for individuals and teams who need to organize chaos and achieve clarity.

## Tech Stack
- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS (using modern `bg-linear-to-r` syntax)
- **Animations:** Framer Motion (motion/react)
- **Icons:** Lucide React
- **Authentication:** NextAuth.js
- **UI Components:** Radix UI / Shadcn UI (Avatar, Dropdowns)

## Design System
- **Primary Palette:** Slate-950 (Background), Violet-600 (Primary Action), Indigo-500 (Secondary).
- **Accents:** Emerald-500 (Success/Done), Amber-400 (Warnings/AI insights), Red-500 (Urgent).
- **Typography:** Sans-serif, tracking-tight for headings, high-readability for task lists.
- **Visual Style:** Glassmorphism, deep shadows, violet/indigo glows, and subtle border-glow effects.

## Core Components
1. **Navigation:** Dynamic navbar with Auth integration and mobile-responsive menu.
2. **Hero:** High-impact value proposition with background ambience.
3. **Product Mockup:** Responsive dashboard visualization showing real-world task scenarios.
4. **AI Simulator:** Interactive component demonstrating natural language processing to task conversion.
5. **Feature Grid:** Icon-based grid explaining USP (Unique Selling Points).
6. **Stat Section:** Social proof and performance metrics.
7. **Testimonials:** "Wall of Love" featuring user feedback.
8. **Pricing:** Multi-tier billing system with monthly/yearly toggles.
9. **Footer:** Comprehensive site navigation and social links.
