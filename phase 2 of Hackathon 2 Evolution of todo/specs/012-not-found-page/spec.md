# Not Found Page Specification

## Overview
Create a custom 404 not-found page for the Next.js application that displays a "Genie Not Found" message with the lamp logo, using proper theme colors.

## Requirements
- Display "Genie Not Found" as the main heading
- Include the lamp logo (from /logo.png) prominently
- Use theme colors: violet gradient for text on dark slate background
- Centered layout for better user experience
- Responsive design for different screen sizes
- Follow Next.js conventions for not-found pages

## Technical Details
- File location: frontend/src/app/not-found.tsx
- Use Next.js Image component for logo
- Apply Tailwind CSS classes for styling
- Ensure accessibility with proper alt text

## Acceptance Criteria
- Page renders when accessing non-existent routes
- Logo displays correctly
- Text styling matches app theme
- Layout is centered and visually appealing
